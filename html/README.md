# Zion Criative — Versão estática

Site em HTML, CSS e JavaScript puro. Nenhuma dependência.

## Como usar

Basta abrir `index.html` no navegador, ou servir a pasta com qualquer servidor estático:

```bash
npx serve .
# ou
python3 -m http.server 8000
```

## Arquivos

- `index.html` — estrutura da página
- `styles.css` — estilos (design system, layout, responsividade)
- `script.js` — renderização dos cards e conteúdo dinâmico
- `assets/` — imagens (hero e portfólio)
